# Craft CMS for Fish Shell

## Install

### Using [Fisher](https://github.com/jorgebucaran/fisher)

```fish
fisher add  dwhoban/fish-craftcms
```

## Usage

Craft CMS CLI completions and local function.

```fish
craft
```
